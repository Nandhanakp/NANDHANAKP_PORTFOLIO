
# 💻 Nandhana KP | Portfolio Website

Welcome to my personal portfolio website! This responsive site showcases who I am, my technical skills, achievements, and how you can connect with me. Built using HTML, CSS, and JavaScript, it highlights my interest in front-end development and community involvement.

## 🌐 Live Preview

You can open `index.html` in any modern browser to see the site in action.

---

## 📁 Project Structure

portfolio/
├── index.html # Main HTML file
├── style.css # Custom styles
├── script.js # Basic JS interactions
├── profile.jpg # Profile picture
└── resume.pdf # Downloadable resume


---

## ✨ Features

- Clean, aesthetic design with smooth layout
- Mobile responsive
- Social media links with hover effects
- Resume download button with alert
- Font Awesome icons for social profiles
- Sections include:
  - Introduction
  - About Me
  - Key Skills
  - Social Media / Contact

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Font Awesome 6](https://fontawesome.com/) for icons

---

## 🚀 Getting Started

To run this website locally:

1. Clone the repo or download the ZIP.
2. Ensure all files (`index.html`, `style.css`, `script.js`, `profile.jpg`, `resume.pdf`) are in the same folder.
3. Open `index.html` in a browser.
